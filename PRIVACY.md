# Privacy Policy

Comment Overlay は、Discord のコメントを Web ページ上に表示するためのツールです。

## Chrome Extension

拡張機能は以下の情報を Chrome の拡張機能ストレージに保存します。

- 表示設定
- 文字色、速度、レーン数などの見た目設定

拡張機能は、表示中の Web ページの本文や閲覧履歴を収集しません。オーバーレイを表示するため、`http://*/*` と `https://*/*` のページに content script を注入します。

## Relay Server

中継サーバーは、Discord Bot が受け取ったコメントを、ペアリング済みのブラウザへ WebSocket で転送します。

転送される情報:

- コメント本文
- Discord 上の表示名
- Discord メッセージ ID
- 表示用の色

現在の実装では、コメントやペアリング情報をデータベースに保存しません。ペアリング状態はサーバープロセスのメモリ上で管理され、サーバー再起動時に消えます。

## Logs

標準のサーバーログには起動状態やエラーが出力されます。運用環境の設定によっては、管理者がログを確認できる場合があります。

## Third Parties

このツールは Discord API を利用します。Discord 上のデータの扱いは Discord の規約とプライバシーポリシーに従います。
